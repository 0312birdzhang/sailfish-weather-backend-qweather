# 旗鱼天气和风天气后端实现

> 大部分是AI写的，存在问题很正常，欢迎提交Pull Request。
> 

## 注册 API

访问 https://dev.qweather.com/ 注册和风天气开放平台账号，创建一个 WebAPI 类型的应用以获取 KEY。

## 配置格式

用户需要提供 API 域名和 KEY，用**英文逗号**分隔。

API HOST 通过 https://console.qweather.com/setting 进入查看

KEY 通过 https://console.qweather.com/project?lang=zh 进入查看

示例：`abcdefg.re.qweatherapi.com,abcdfeasdasdasdasdasdad`

## 功能特性

- 实时天气数据
- 24 小时逐小时预报
- 7 天逐日预报
- 位置搜索功能

## 安装

本插件为 Sailfish OS 应用，可通过 RPM 包安装。

## 使用说明

1. 安装插件
2. 打开 Sailfish Weather 应用
3. 在天气源设置中选择「和风天气」
4. 输入 API 域名和 KEY（用英文逗号分隔）
